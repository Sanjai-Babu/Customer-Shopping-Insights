# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using SQL, Python, and Power BI. The objective is to uncover customer purchasing patterns, identify key sales trends, and generate actionable business insights through data analytics and visualization.

## Business Problem

Retail businesses need to understand customer behavior to improve sales performance, customer retention, and product recommendations. This project explores customer demographics, purchasing habits, payment preferences, and product category performance to support data-driven decision-making.

## Dataset

The dataset contains customer shopping records, including:

* Customer demographics
* Product categories
* Purchase amounts
* Payment methods
* Customer ratings
* Purchase frequency
* Shopping preferences

## Project Workflow

### 1. Data Collection

Imported customer shopping behavior data from CSV format.

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats
* Validated data quality

### 3. Exploratory Data Analysis (EDA)

Performed analysis to identify:

* Customer demographics distribution
* Product category performance
* Spending patterns
* Purchase frequency trends
* Customer satisfaction metrics

### 4. SQL Analysis

Used SQL queries to:

* Aggregate sales data
* Calculate KPIs
* Analyze customer segments
* Generate business insights

### 5. Power BI Dashboard

Created interactive dashboards featuring:

* Total Sales
* Average Purchase Value
* Product Category Analysis
* Customer Demographics
* Payment Method Distribution
* Customer Rating Analysis
* Purchase Frequency Insights

## Key Insights

* Identified top-performing product categories.
* Analyzed customer spending behavior across demographics.
* Evaluated preferred payment methods.
* Discovered trends in purchase frequency and customer engagement.
* Highlighted opportunities for targeted marketing strategies.
